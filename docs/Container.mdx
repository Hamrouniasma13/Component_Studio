# Container

`Container` is the main component when it comes to wrap the content on a reasonable size. It's size depend on the media :

- _mobile_ : `100%` of the screen
- _tablet_ : `720px`
- _laptop_ : `960px`
- _desktop_ : `1140px`

It always come with `15px` padding on the left and on the right, and horizontally centered.

## Article

`Container` can't be customized, but you can use it's smaller version with the `article` property, which will make it not larger than `798px`.
