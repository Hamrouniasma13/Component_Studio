Each row fo the table can take theses properties, but you may want to keep consistency on the same table.

- **title**: label in the row.
- **link**: target when you click on the row. It automatically adds a border and an icon.
- **value**: value display at the end of the line.
- **simulation**: appear on the left of the value, strike through.
- **pill**: object for pill specification, before the label.
- **dot**: Boolean to disable the `pill` and add a dot instead.
