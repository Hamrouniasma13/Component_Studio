# NavigationBanner

The title is a `h1`.

Colors :

- For the background, provide the color as a css value _(hexadecimal, rgb, keyword, etc...)_
- For the color of the title and the slider elements (arrows, dots), we use `theme.colors.white` or `theme.colors.text`, depends on the backgroundColor props.
- As the action is an element, we don't set its color
- For each cards, the background is always `#fff`, and the text color is `theme.colors.text`

## Card title as element

Cards can take title as en element, allowing you to customize it with `<strong>` or `<em>` tags for example.  
**Don't use `<a>` in the title**
