## Card content as element

- It's not mandatory to have 3 or more cards. You can have less, but think about you design first.
- `title` and `action` are optionnal props.
- You use element as card content, allowing more customisation on this specific stuff.

<Playground fullBleed noPadding>

    cards={[
      {
        image: {
