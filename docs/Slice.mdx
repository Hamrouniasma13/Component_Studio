# Slice

`Slice` is the main component used on slices.
It puts a `70px` top margin and a `100px` bottom margin to separate each slice with the same space.
It also put a [Container](/container), so the content automatically fits the viewport, and is centered.
As children, `Slice` take any component.

As each page can have multiple slices, each title is an `h2`, with the size of an `h1`.
The title is optionnal.

The `actions` prop takes an array of component that will be render on the bottom right, below the content.
Even if you can put any component, we advise to use [Button](/button) or [SimpleButton](/simple-button)
The actions are optionnal.
