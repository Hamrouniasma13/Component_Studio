# Thumbnails

Semantic

- Main title is an `h2`
- Cards title are `h3` tag
- Content are wrapped in a span

Colors :

- `theme.colors.white` for texts
- `#333` with `opacity: 0.4` for caption background

The hover effect is disabled if there is url for a card.
