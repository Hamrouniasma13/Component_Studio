# Theming

Theming allows you to fully customize the display of a website.
On our side, you'll be able to only change some colors.

## Theme Provider

_For now_, you **have to** wrap all the components inside the `ThemeProvider` we provide

<Playground>
  <ThemeProvider>My themed App</ThemeProvider>
</Playground>

It will put the default theme in place, which is mandatory for our components to render.

## Change colors

You can change the colors by giving your theme to the `ThemeProvider`

<Playground>
  <ThemeProvider
    theme={{
      colors: {
        primary: '#938293',
      },
    }}
  >
    My custom themed App <Button href="/">Custom button</Button>
  </ThemeProvider>
</Playground>

## Global style

You can also use our `GlobalStyles` component, which will reset some margins and paddings, remove some browser style, and adjust the box-sizing property of every element.
It should be placed once in your app.

<Playground>
  <div>
    <GlobalStyles />
    My App
  </div>
</Playground>
